# HB_Elise

## About This Repository

This is a public repository for the HB_Elise project.

## Repository Security

**Important**: This repository is public, which means anyone can view and clone the code. However, **only authorized collaborators can make direct changes**.

For detailed information about repository permissions and security, please read [REPOSITORY_PERMISSIONS.md](REPOSITORY_PERMISSIONS.md).

### Key Points:
- ✅ Public = Anyone can VIEW your code
- ❌ Public ≠ Anyone can CHANGE your code
- All changes require your approval through pull requests
- You have full control over what gets merged

## Contributing

If you'd like to contribute to this project:

1. Fork this repository
2. Make your changes in your fork
3. Submit a pull request
4. Wait for review and approval

**Note**: Pull requests will be reviewed before being merged. This ensures the repository owner maintains control over all changes.
